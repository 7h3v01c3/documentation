1. Navigate to the data directory.

	??? abstract "Windows"
		```
		C:\Users\[YourUsername]\AppData\Roaming\BlocknetDX\
		```
		Or paste `%appdata%\Roaming\BlocknetDX\` into the file explorer path field.

	??? abstract "MacOS"
		```
		~/Library/Application Support/BlocknetDX/
		```

	??? abstract "Linux"
		```
		~/.blocknetdx/
		```
